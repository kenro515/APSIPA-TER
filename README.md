# APSIPA-TER

This code is the implementation of Text Emotion Recognition (TER) with linguistic features.
The network model is BERT with a pretrained model parameter.

## How to use
0. Preprocess your files
1. Edit hyper_param.yaml
2. Run main.py
> python3 main.py
